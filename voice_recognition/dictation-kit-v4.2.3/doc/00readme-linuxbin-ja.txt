Linux バイナリパッケージ of Julius-4.2.3
=========================================

このアーカイブは Julius-4.2.3 のコンパイル済み実行バイナリと
ライブラリのほか，ヘッダファイル，サンプルコード，マニュアル等が
含まれています．

Julius rev.4.2.3 のリリース情報は "00readme-julius-ja.txt" をご覧くだ
さい．"Release-ja.txt" には前バージョンからの詳細な差分の説明があります．

最新版は常に Web 上から入手できます．
また，他にも多くの情報が Web から取得できますのでご参照ください．

	http://julius.sourceforge.jp/


インストール方法
==================

"./run_install.sh" を実行してください．インストールするディレクトリを
聞かれるので答えてください．デフォルトでは "/usr/local" です．
その後 確認に対して "y" と答えると必要なファイルのコピーが行われ，
インストールが行われます．

----------------------------------------------------------------------
Julius および関連ツールの実行ファイルは "bin" に含まれています．
コアライブラリである JuliusLib のヘッダが "include" に，スタティック
ライブラリが "lib" に含まれています．

"julius-simple" ディレクトリには JuliusLib を利用するサンプルプログラ
ムが含まれています．これはJuliusの簡易版であり，同じ様式のコマンドオプ
ションに従って音声認識を起動し，認識結果を標準出力に出力します．
以下のコマンドでコンパイルできますので，参考にしてください．

      % cd julius-simple
      % make

"plugin-sample" には，Julius用プラグイン開発のためのサンプルプログラム
が含まれています（内容はソースアーカイブに含まれているものと同じです）．
"make" でコンパイルできます．詳細は，ディレクトリ内の 00readme-ja.txt
および Juliusbook の11章を参考にしてください．
